A Decentralized Name System for the New Internet

We aim to create a decentralized autonomous organization (DAO) that will replace the DNS system and change the economics of domain ownership. Only by incubating a DAO Registrar with its own tokens can we create a lasting self sustaining economy that can replace ICANN. This ability to remove people and governments from the central control of the Internet is paramount to create a lasting uncensored internet.

Not your Grandpa’s Web Naming System

The internet is changing and being used in ways the creators of the original DNS could never have imagined. With Butterfly flexibility and expandability are at its core. It isn’t simply bolting old tech on new applications, it was made with the future in mind.

Token Creation

Each name is itself an ERC-721 token and can easily create ERC-223 tokens. This makes for a powerful token platform with a built in name and directory making them easy to find and manage.

Github Replacement

Git was designed to be decentralized. Using a tool site like github made it easy to find projects, but it’s recent sale to Microsoft is a perfect example of how what was once a safe haven can quickly be taken over by corporate interests. Using Butterfly gives us a better option.

Gaming

The incredibly powerful sub name system opens up an entire ecosystem of functional sub names that can be used as the basis for games and other yet to be invented applications.

Fully Decentralized Dapps

Dapps built on blockchains have huge potential, but most of the time they still have central points of failure. With Butterfly for naming and IPFS to store static assets developers now have a way to create fully decentralized Dapps.

Butterfly is more than a replacement, it is an upgrade

There is no point taking the existing system and just putting it on a blockchain. Butterfly isn’t just DNS on a blockchain, it is a better name system. We looked at where the current system was lacking and what it could look like if it wasn’t held back by legacy technology. We built Butterfly to be the name system the brave new internet needs and deserves.

Websites are constantly being blocked by hostile state actors. With Butterfly your name lives on the blockchain forever and can’t be taken down.

Own Your Own Voice

As a way to showcase the decentralized and censorship resistant qualities of the Butterfly platform we are creating the Butterfly mobile app.  This new app will allow anyone in the world to easily acquire a .HUMAN subdomain name and then use that domain to publish content for others to read.  No person or organization can block your content, censor you, or cancel your account.  You own everything.
